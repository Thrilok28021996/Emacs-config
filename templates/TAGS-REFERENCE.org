* 💡 How to Use Tags Effectively

** Best Practices

1. *Be Consistent*: Use the same tag for similar content
2. *Start Broad*: Use general tags, add specific ones as needed
3. *Limit Tags*: Use 3-7 tags per note for optimal findability
4. *Combine Tags*: Create context with tag combinations
   Example: ~:python:algorithm:learning:~ for Python algorithm learning

5. *Use Hierarchies*: Create tag relationships
   Example: ~:learning:course:python:~

** Tag Syntax

*** File-level Tags (Applies to entire file)
#+begin_example
#+FILETAGS: :tag1:tag2:tag3:
#+end_example

*** Heading-level Tags
#+begin_example
* My Heading :tag1:tag2:
#+end_example

*** Inherited Tags
Child headings inherit parent tags automatically!

** Searching by Tags

*** In Org-agenda
- Press ~m~ for tag search
- Use expressions: ~python+learning~ or ~python|cpp~
- Exclude tags: ~python-archived~

*** With Org-roam
- ~SPC n f~ (org-roam-node-find) then filter by tag
- Search: ~tag:python~
