# Deliberate Practice System

**Goal:** Get better at anything through focused, consistent practice

## How It Works
1. **Pick ONE skill** per week
2. **Practice 25 minutes daily** using templates
3. **Track progress** with simple scores
4. **Review weekly** to improve approach

## Daily Routine (45 min total)

### Skills Focus (30 min)
**Morning (5 min):** Set today's target → [[Templates/Daily Practice]]
**Practice (25 min):** Focused skill building
**Evening (5 min):** Quick reflection

### Insight Generation (15 min)
**Morning (5 min):** Review yesterday's ideas, set exploration focus
**Throughout day:** Capture ideas → [[Templates/Quick Idea]]
**Evening (10 min):** Process captures, note connections

## Weekly Routine (50 min)
**Sunday:** Choose next skill → [[Templates/Skill Sprint]] (20 min)
**Saturday:** Skills review → [[Templates/Weekly Review]] (20 min)
**Sunday:** Insight harvest → [[Templates/Weekly Insight Review]] (30 min)

## The System Rules
1. **One skill at a time** - Focus beats multitasking
2. **Daily practice** - Consistency beats intensity
3. **Track everything** - Numbers don't lie
4. **Weekly adjustment** - Adapt based on what works
5. **Output matters** - Create something with your skill
